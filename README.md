# RIP OTT

Telegram bot for downloading torrents and getting fast HTTPS links.

## Commands

- `/start` - Welcome message
- `/help` - Show help
- `/torrent <magnet>` - Add magnet link
- `/status <id>` - Check torrent status  
- `/download <link>` - Get download link

## Usage

1. Upload `.torrent` file or use `/torrent <magnet_link>`
2. Check status with `/status <torrent_id>`
3. Download files with `/download <file_link>`
